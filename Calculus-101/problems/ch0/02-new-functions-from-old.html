0.2 New Functions from Old0.2 由舊函數造新函數

1 · Concept and Fundamental Knowledge1 · 概念與基礎知識

Most functions you will ever use are built from a few simple ones. There are two ways to build a new function from old ones: combine them with arithmetic (sum, difference, product, quotient) or with composition, and transform a known graph by shifting, reflecting, or stretching it. This section reads transformations directly off graphs, because once you can see how a graph moves you can predict the new function without plotting a single extra point.你會用到的大多數函數，都是由少數幾個簡單函數組合而成。由舊函數造新函數有兩種方式：用四則運算（和、差、積、商）或合成來組合它們，以及把已知圖形平移、反射或伸縮來變換它。本節直接由圖形判讀變換，因為一旦看得出圖形如何移動，就能在不多畫任何一點的情況下預測新函數。

Definition · Composition of functions定義 · 函數合成

Given two functions \(f\) and \(g\), their composition \(f\circ g\) is the function that applies \(g\) first and then \(f\):給定兩個函數 \(f\) 與 \(g\)，它們的合成 \(f\circ g\) 是先做 \(g\)、再做 \(f\) 的函數：

\[ (f\circ g)(x)=f\bigl(g(x)\bigr). \]

Read it inside-out: take the input \(x\), feed it to \(g\), then feed the result \(g(x)\) to \(f\). The order matters, and the domain shrinks to those \(x\) for which \(g(x)\) is an allowed input of \(f\).由內而外讀：把輸入 \(x\) 餵給 \(g\)，再把結果 \(g(x)\) 餵給 \(f\)。順序很重要，且定義域會縮小到那些使 \(g(x)\) 成為 \(f\) 合法輸入的 \(x\)。

Transformations are the other half of the story. Every transformation is one small change to either the output of a function (an operation written outside \(f\)) or its input (an operation written inside \(f\)). The three you will read most often are:變換是另一半的故事。每個變換都是對函數的輸出（寫在 \(f\) 外側的運算）或輸入（寫在 \(f\) 內側的運算）所做的一個小改變。最常判讀的三種是：

Key terms · transformations of a graph重要名詞 · 圖形的變換

  • Vertical shift. \(y=f(x)+c\) slides the whole graph up by \(c\) (or down if \(c\lt 0\)). The domain is unchanged; the range slides by \(c\).垂直平移。\(y=f(x)+c\) 把整個圖形上移 \(c\)（若 \(c\lt 0\) 則下移）。定義域不變；值域平移 \(c\)。
  • Horizontal shift. \(y=f(x-c)\) slides the whole graph right by \(c\) (or left if \(c\lt 0\)). Notice the inside sign is opposite to the motion. The range is unchanged; the domain slides by \(c\).水平平移。\(y=f(x-c)\) 把整個圖形右移 \(c\)（若 \(c\lt 0\) 則左移）。注意內側的正負號與移動方向相反。值域不變；定義域平移 \(c\)。
  • Vertical scaling. \(y=a\,f(x)\) stretches the graph vertically when \(a\gt 1\) and compresses it toward the \(x\)-axis when \(0\lt a\lt 1\). The domain is unchanged; every height is multiplied by \(a\).垂直伸縮。\(y=a\,f(x)\) 在 \(a\gt 1\) 時把圖形垂直拉伸，在 \(0\lt a\lt 1\) 時朝 \(x\) 軸壓縮。定義域不變；每個高度都乘上 \(a\)。
  • Reflection and symmetry. \(y=-f(x)\) flips the graph across the \(x\)-axis and \(y=f(-x)\) flips it across the \(y\)-axis. A function is even if \(f(-x)=f(x)\) (mirror-symmetric about the \(y\)-axis) and odd if \(f(-x)=-f(x)\) (symmetric about the origin).反射與對稱。\(y=-f(x)\) 把圖形對 \(x\) 軸翻折，\(y=f(-x)\) 把圖形對 \(y\) 軸翻折。若 \(f(-x)=f(x)\)（對 \(y\) 軸鏡像對稱）則函數為偶函數；若 \(f(-x)=-f(x)\)（對原點對稱）則為奇函數。

Rule · outside acts vertically, inside acts horizontally規則 · 外側管垂直、內側管水平

An operation written outside \(f\) (on the output, like \(+c\) or \(\times a\)) changes the graph vertically and in the natural direction. An operation written inside \(f\) (on the input \(x\)) changes the graph horizontally and in the reversed sense, so \(f(x-c)\) moves right and \(f(-x)\) flips left-right. When in doubt, ask one question: is the change touching the output or the input?寫在 \(f\) 外側的運算（作用於輸出，如 \(+c\) 或 \(\times a\)）會垂直且依自然方向改變圖形。寫在 \(f\) 內側的運算（作用於輸入 \(x\)）會水平且依相反方向改變圖形，所以 \(f(x-c)\) 右移、\(f(-x)\) 左右翻折。不確定時只問一句：這個改變動到的是輸出還是輸入？

2 · Worked Examples2 · 範例（詳解）

Throughout these three examples the base function \(f\) is the one with corner points \((0,2)\), \((2,6)\), \((4,4)\): it rises from \((0,2)\) to a peak at \((2,6)\), then falls to \((4,4)\), so \(\text{domain}(f)=[0,4]\) and \(\text{range}(f)=[2,6]\). Each example draws a single transformation of \(f\). Read the new graph, then check it against the rule.這三個範例中，基底函數 \(f\) 是以轉角點 \((0,2)\)、\((2,6)\)、\((4,4)\) 構成的函數：由 \((0,2)\) 上升到頂點 \((2,6)\)，再下降到 \((4,4)\)，故 \(\text{domain}(f)=[0,4]\)、\(\text{range}(f)=[2,6]\)。每個範例只畫 \(f\) 的一個變換。請判讀新圖形，再對照規則。

Example 1 · A vertical shift範例 1 · 垂直平移

The graph below is \(g(x)=f(x)+1\). Every point of the base \(f\) has been lifted straight up by \(1\). Read its domain, its range, and the value \(g(3)\).下圖是 \(g(x)=f(x)+1\)。基底 \(f\) 的每個點都被向上抬升 \(1\)。請讀出它的定義域、值域，以及函數值 \(g(3)\)。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
Domain定義域 \([0,4]\) A vertical shift moves nothing left or right, so the domain is the same as the base \(f\).垂直平移不會左右移動，故定義域與基底 \(f\) 相同。
Range值域 \([3,7]\) The base range \([2,6]\) is lifted by \(1\): the low \(2\) becomes \(3\), the peak \(6\) becomes \(7\).基底值域 \([2,6]\) 被抬升 \(1\)：最低的 \(2\) 變成 \(3\)，頂點 \(6\) 變成 \(7\)。
\(g(3)\) \(6\) Read \(f(3)=5\) on the base, then add \(1\). On the graph, go up from \(x=3\) to the curve and across: the height is \(6\).先在基底讀出 \(f(3)=5\)，再加 \(1\)。在圖上由 \(x=3\) 上移到曲線再水平移過去：高度為 \(6\)。

A vertical shift is an outside operation: it adds \(c\) to the output. The shape and the domain do not change at all; only the range slides, by exactly \(c\).垂直平移是外側運算：它把 \(c\) 加到輸出。形狀與定義域完全不變；只有值域平移，且恰好平移 \(c\)。

Example 2 · A horizontal shift範例 2 · 水平平移

The next graph is \(p(x)=f(x-1)\). The \(-1\) sits inside \(f\), on the input. Read its domain, its range, and the value \(p(4)\), and watch which way the graph actually moves.下一張圖是 \(p(x)=f(x-1)\)。這個 \(-1\) 在 \(f\) 內側、作用於輸入。請讀出它的定義域、值域，以及函數值 \(p(4)\)，並注意圖形實際往哪個方向移動。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
Domain定義域 \([1,5]\) The base domain \([0,4]\) slides right by \(1\). The \(-1\) inside \(f\) moves the graph the opposite way to its sign: right, not left.基底定義域 \([0,4]\) 右移 \(1\)。\(f\) 內側的 \(-1\) 使圖形朝與其正負號相反的方向移動：是右移，不是左移。
Range值域 \([2,6]\) A horizontal shift never changes the outputs, so the range is exactly the base range.水平平移不會改變輸出，故值域與基底值域完全相同。
\(p(4)\) \(5\) By definition \(p(4)=f(4-1)=f(3)=5\). On the graph, go up from \(x=4\) to the curve: the height is \(5\).依定義 \(p(4)=f(4-1)=f(3)=5\)。在圖上由 \(x=4\) 上移到曲線：高度為 \(5\)。

Notice · inside shifts go the opposite way注意 · 內側平移方向相反

\(f(x-1)\) moves the graph right, even though the sign is negative. The reason: to get the same height the base had at input \(3\), the new function must be fed \(x=4\), one unit further right. An inside operation always undoes its sign.\(f(x-1)\) 把圖形右移，即使符號是負的。原因：要得到基底在輸入 \(3\) 處的高度，新函數必須餵入 \(x=4\)，也就是再往右一格。內側運算總是把它的正負號反過來。

Example 3 · A vertical compression範例 3 · 垂直壓縮

This graph is \(q(x)=\tfrac{1}{2}\,f(x)\). Every output of the base is multiplied by \(\tfrac{1}{2}\). Read its domain, its range, and the value \(q(1)\).這張圖是 \(q(x)=\tfrac{1}{2}\,f(x)\)。基底的每個輸出都乘上 \(\tfrac{1}{2}\)。請讀出它的定義域、值域，以及函數值 \(q(1)\)。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
Domain定義域 \([0,4]\) Vertical scaling is an outside operation, so it leaves the inputs alone; the domain matches the base \(f\).垂直伸縮是外側運算，不動輸入；定義域與基底 \(f\) 相同。
Range值域 \([1,3]\) Every height of the base range \([2,6]\) is halved: \(2\to 1\) and \(6\to 3\).基底值域 \([2,6]\) 的每個高度都減半：\(2\to 1\)、\(6\to 3\)。
\(q(1)\) \(2\) Read \(f(1)=4\) on the base, then halve it: \(\tfrac{1}{2}\cdot 4=2\).先在基底讀出 \(f(1)=4\)，再取一半：\(\tfrac{1}{2}\cdot 4=2\)。

A vertical scaling by \(a\) multiplies every output by \(a\), pulling the graph toward the \(x\)-axis when \(0\lt a\lt 1\). The domain is untouched; the range is scaled by the same factor.以 \(a\) 做垂直伸縮會把每個輸出乘上 \(a\)，在 \(0\lt a\lt 1\) 時把圖形朝 \(x\) 軸拉近。定義域不受影響；值域以相同倍率縮放。

3 · Exercises3 · 練習

Exercise 1 · A vertical shift練習 1 · 垂直平移

Now it is your turn. Let the base function \(b\) have corner points \((0,2)\), \((2,6)\), \((4,4)\). The graph below is a single transformation \(h\) of \(b\), all of whose points have moved straight down.換你試試。設基底函數 \(b\) 的轉角點為 \((0,2)\)、\((2,6)\)、\((4,4)\)。下圖是 \(b\) 的一個變換 \(h\)，其所有點都向正下方移動。

Using the graph of \(h\) and the base \(b\), answer all five parts:利用 \(h\) 的圖形與基底 \(b\)，回答以下五個小題：

  1. State the domain of \(h\) in interval notation.以區間記號寫出 \(h\) 的定義域。
  2. State the range of \(h\) in interval notation.以區間記號寫出 \(h\) 的值域。
  3. Find \(h(1)\).求 \(h(1)\)。
  4. Find \(h(3)\).求 \(h(3)\)。
  5. Write \(h\) as a transformation of \(b\) in the form \(h(x)=b(x)+c\), and say where the base peak \((2,6)\) has moved to.將 \(h\) 以 \(h(x)=b(x)+c\) 的形式寫成 \(b\) 的變換，並說明基底頂點 \((2,6)\) 移到了何處。
Hint:提示： Every point of \(b\) dropped by the same amount. Compare a base height to the matching height on \(h\) (for example the peak) to find \(c\). A vertical shift leaves the domain alone.\(b\) 的每個點都下降相同的量。把基底某個高度與 \(h\) 上對應的高度相比（例如頂點）即可求出 \(c\)。垂直平移不改變定義域。
Show full solution顯示完整解答

(1) Domain. A vertical shift moves no point left or right, so \(h\) is defined on the same inputs as \(b\), from the filled left end \(x=0\) to the filled right end \(x=4\):(1) 定義域。垂直平移不左右移動任何點，故 \(h\) 的定義輸入與 \(b\) 相同，由實心左端 \(x=0\) 到實心右端 \(x=4\)：

\[ \text{domain}(h)=[0,4]. \]

(2) Range. The lowest point of \(h\) is the left end at height \(0\); the highest is the peak at height \(4\). Every height between is reached, so:(2) 值域。\(h\) 的最低點是左端高度 \(0\)，最高是頂點高度 \(4\)。其間每個高度都被取到，故：

\[ \text{range}(h)=[0,4]. \]

(3) \(h(1)\). On the base, \(b(1)=4\) (halfway up from \((0,2)\) to \((2,6)\)). Reading \(h\) at \(x=1\) gives the height halfway from \((0,0)\) to \((2,4)\):(3) \(h(1)\)。在基底上 \(b(1)=4\)（由 \((0,2)\) 到 \((2,6)\) 的中點）。在 \(x=1\) 讀 \(h\)，得到由 \((0,0)\) 到 \((2,4)\) 的中點高度：

\[ h(1)=2. \]

(4) \(h(3)\). Now \(x=3\) is halfway down from the peak \((2,4)\) to \((4,2)\):(4) \(h(3)\)。此時 \(x=3\) 位於由頂點 \((2,4)\) 下降到 \((4,2)\) 的中點：

\[ h(3)=3. \]

(5) As a transformation. Each base height has dropped by \(2\) (the peak \(6\) became \(4\)), so:(5) 寫成變換。每個基底高度都下降了 \(2\)（頂點 \(6\) 變成 \(4\)），故：

\[ h(x)=b(x)-2,\qquad (2,6)\ \mapsto\ (2,4). \]

Sense check. Subtracting \(2\) from the base range \([2,6]\) gives \([0,4]\), matching part (2). The readings \(h(1)=2\) and \(h(3)=3\) are exactly \(b(1)=4\) and \(b(3)=5\) each minus \(2\), and both lie inside \([0,4]\). Everything is consistent.合理性檢查。把基底值域 \([2,6]\) 減 \(2\) 得 \([0,4]\)，與第 (2) 題相符。讀值 \(h(1)=2\)、\(h(3)=3\) 恰為 \(b(1)=4\)、\(b(3)=5\) 各減 \(2\)，且都落在 \([0,4]\) 內。一切一致。

Exercise 2 · A vertical stretch練習 2 · 垂直拉伸

Here the base function \(c\) has corner points \((0,2)\), \((2,4)\), \((4,3)\). The graph below is a single transformation \(k\) of \(c\) in which every height has grown. Read carefully: this is an outside operation, not a shift.這裡基底函數 \(c\) 的轉角點為 \((0,2)\)、\((2,4)\)、\((4,3)\)。下圖是 \(c\) 的一個變換 \(k\)，其中每個高度都變大了。請仔細判讀：這是外側運算，不是平移。

Using the graph of \(k\) and the base \(c\), answer all five parts:利用 \(k\) 的圖形與基底 \(c\)，回答以下五個小題：

  1. State the domain of \(k\) in interval notation.以區間記號寫出 \(k\) 的定義域。
  2. State the range of \(k\) in interval notation.以區間記號寫出 \(k\) 的值域。
  3. Find \(k(1)\).求 \(k(1)\)。
  4. Find \(k(3)\).求 \(k(3)\)。
  5. Write \(k\) as a transformation of \(c\) in the form \(k(x)=a\,c(x)\), and say what happens to the domain.將 \(k\) 以 \(k(x)=a\,c(x)\) 的形式寫成 \(c\) 的變換，並說明定義域有何變化。
Hint:提示： Compare a base height to the matching height on \(k\) (the peak \(c(2)=4\) became \(8\)) to find the factor \(a\). A vertical scaling is an outside operation, so the domain does not move.把基底某個高度與 \(k\) 上對應的高度相比（頂點 \(c(2)=4\) 變成 \(8\)）即可求出倍率 \(a\)。垂直伸縮是外側運算，故定義域不動。
Show full solution顯示完整解答

(1) Domain. A vertical scaling is an outside operation and leaves the inputs untouched, so \(k\) is defined from \(x=0\) to \(x=4\):(1) 定義域。垂直伸縮是外側運算，不動輸入，故 \(k\) 的定義由 \(x=0\) 到 \(x=4\)：

\[ \text{domain}(k)=[0,4]. \]

(2) Range. The lowest point of \(k\) is the left end at height \(4\); the highest is the peak at height \(8\). So:(2) 值域。\(k\) 的最低點是左端高度 \(4\)，最高是頂點高度 \(8\)。故：

\[ \text{range}(k)=[4,8]. \]

(3) \(k(1)\). On the base, \(c(1)=3\) (halfway up from \((0,2)\) to \((2,4)\)). Reading \(k\) at \(x=1\) is halfway up from \((0,4)\) to \((2,8)\):(3) \(k(1)\)。在基底上 \(c(1)=3\)（由 \((0,2)\) 到 \((2,4)\) 的中點）。在 \(x=1\) 讀 \(k\)，是由 \((0,4)\) 到 \((2,8)\) 的中點：

\[ k(1)=6. \]

(4) \(k(3)\). Now \(x=3\) is halfway down from the peak \((2,8)\) to \((4,6)\):(4) \(k(3)\)。此時 \(x=3\) 位於由頂點 \((2,8)\) 下降到 \((4,6)\) 的中點：

\[ k(3)=7. \]

(5) As a transformation. Each base height has doubled (the peak \(4\) became \(8\), the end \(2\) became \(4\)), so:(5) 寫成變換。每個基底高度都變成兩倍（頂點 \(4\) 變成 \(8\)，端點 \(2\) 變成 \(4\)），故：

\[ k(x)=2\,c(x), \]

and the domain is unchanged at \([0,4]\): an outside operation never moves the inputs.且定義域不變，仍為 \([0,4]\)：外側運算永遠不動輸入。

Sense check. Doubling the base range \([2,4]\) gives \([4,8]\), matching part (2). The readings \(k(1)=6\) and \(k(3)=7\) are exactly \(c(1)=3\) and \(c(3)=3.5\) each times \(2\), and both lie inside \([4,8]\). Everything is consistent.合理性檢查。把基底值域 \([2,4]\) 加倍得 \([4,8]\)，與第 (2) 題相符。讀值 \(k(1)=6\)、\(k(3)=7\) 恰為 \(c(1)=3\)、\(c(3)=3.5\) 各乘 \(2\)，且都落在 \([4,8]\) 內。一切一致。
