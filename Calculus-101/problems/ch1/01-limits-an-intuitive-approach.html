1.1 Limits (An Intuitive Approach)1.1 極限（直觀方法）

1 · Concept and Fundamental Knowledge1 · 概念與基礎知識

Definition · Limit near a point定義 · 點附近的極限

We write \(\lim_{x\to a}f(x)=L\) when the output of \(f(x)\) gets close to \(L\) as the input \(x\) gets close to \(a\). The key idea is near, not at: the limit describes the trend around \(x=a\), even if \(f(a)\) is missing or different.當輸入 \(x\) 靠近 \(a\) 時，若函數輸出 \(f(x)\) 會靠近 \(L\)，就寫成 \(\lim_{x\to a}f(x)=L\)。關鍵是看「附近」，不是看「剛好在該點」：極限描述 \(x=a\) 周圍的趨勢，即使 \(f(a)\) 不存在或與趨勢不同也可以。

A graph gives this idea directly. Move toward \(x=a\) from the left and from the right, and read the height your curve approaches. If both sides approach the same height, that common height is the two-sided limit.圖形可以直接呈現這個概念。從左邊與右邊都往 \(x=a\) 靠近，讀出曲線高度分別趨近哪個值。若兩側都趨近同一高度，該共同高度就是雙邊極限。

Key terms · one-sided and two-sided limits重要名詞 · 單邊與雙邊極限

  • Left-hand limit. \(\lim_{x\to a^{-}}f(x)\) means approach \(a\) using values \(x\lt a\).左極限。\(\lim_{x\to a^{-}}f(x)\) 表示只用 \(x\lt a\) 的值往 \(a\) 靠近。
  • Right-hand limit. \(\lim_{x\to a^{+}}f(x)\) means approach \(a\) using values \(x\gt a\).右極限。\(\lim_{x\to a^{+}}f(x)\) 表示只用 \(x\gt a\) 的值往 \(a\) 靠近。
  • Two-sided limit. \(\lim_{x\to a}f(x)\) exists only when both one-sided limits exist and are equal.雙邊極限。\(\lim_{x\to a}f(x)\) 只有在左右極限都存在且相等時才存在。
  • Function value at the point. \(f(a)\) is the plotted value at \(x=a\), shown by a filled dot if defined.該點函數值。\(f(a)\) 是 \(x=a\) 處真正畫出的函數值，若有定義通常以實心點表示。

Rule · When a two-sided limit exists規則 · 雙邊極限存在的條件

For a point \(x=a\), first read \(\lim_{x\to a^{-}}f(x)\), then \(\lim_{x\to a^{+}}f(x)\). If the two values match, then \(\lim_{x\to a}f(x)\) exists and equals that value. If they differ, the two-sided limit does not exist.在 \(x=a\) 這一點，先讀 \(\lim_{x\to a^{-}}f(x)\)，再讀 \(\lim_{x\to a^{+}}f(x)\)。若兩值相同，則 \(\lim_{x\to a}f(x)\) 存在且等於該值；若兩值不同，則雙邊極限不存在。

2 · Worked Examples2 · 範例（詳解）

Example 1 · Continuous at the point範例 1 · 在該點連續

This graph of \(f\) is a single straight segment. Read \(\lim_{x\to 2^{-}}f(x)\), \(\lim_{x\to 2^{+}}f(x)\), \(\lim_{x\to 2}f(x)\), and \(f(2)\).下圖的 \(f\) 是一條連續直線段。請讀出 \(\lim_{x\to 2^{-}}f(x)\)、\(\lim_{x\to 2^{+}}f(x)\)、\(\lim_{x\to 2}f(x)\) 與 \(f(2)\)。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
Left limit左極限 \(\lim_{x\to 2^{-}}f(x)=3\) Following the line from \(x\lt 2\), the height approaches 3.沿著 \(x\lt 2\) 的線段靠近 2，高度趨近 3。
Right limit右極限 \(\lim_{x\to 2^{+}}f(x)=3\) Following from \(x\gt 2\), the same line also approaches height 3.由 \(x\gt 2\) 一側靠近，仍是同一條線，所以也趨近高度 3。
Two-sided雙邊 \(\lim_{x\to 2}f(x)=3\) Left and right limits match, so the two-sided limit exists and equals 3.左右極限相同，因此雙邊極限存在且等於 3。
\(f(2)\) \(3\) At \(x=2\) the filled point is also at height 3.在 \(x=2\) 的實心點也位於高度 3。

This is the plain case: the trend and the actual value agree.這是最基本情況：趨勢值與實際函數值一致。

Example 2 · A removable hole範例 2 · 可去空洞

Now \(g\) follows the same line trend, but at \(x=2\) the point on the line is open and a different filled point is plotted above. Read the limits and \(g(2)\).現在 \(g\) 仍沿著同一條線的趨勢，但在 \(x=2\) 時線上的點是空心，並在上方畫了另一個實心點。請讀出極限與 \(g(2)\)。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
Left limit左極限 \(\lim_{x\to 2^{-}}g(x)=3\) From the left, the line approaches the open point at height 3.由左側靠近時，曲線趨近空心點高度 3。
Right limit右極限 \(\lim_{x\to 2^{+}}g(x)=3\) From the right, the same line also approaches height 3.由右側靠近時，仍沿同一直線趨近高度 3。
Two-sided雙邊 \(\lim_{x\to 2}g(x)=3\) Both one-sided limits match at 3.左右極限都等於 3。
\(g(2)\) \(5\) The filled dot at \(x=2\) is at height 5, so the function value is 5.\(x=2\) 的實心點在高度 5，因此函數值是 5。

Notice · limit value can differ from function value注意 · 極限值可與函數值不同

The limit tracks where nearby points go. The value \(g(2)\) depends only on the filled dot at \(x=2\).極限看的是附近點往哪裡去。\(g(2)\) 只由 \(x=2\) 的實心點決定。

Example 3 · A jump at the point範例 3 · 該點發生跳躍

In this graph of \(p\), the left branch approaches one height at \(x=2\), while the right branch approaches a different height. Read the one-sided limits and decide whether \(\lim_{x\to 2}p(x)\) exists.在這張 \(p\) 的圖中，左支在 \(x=2\) 附近趨近一個高度，右支趨近另一個高度。請讀出左右極限並判斷 \(\lim_{x\to 2}p(x)\) 是否存在。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
Left limit左極限 \(\lim_{x\to 2^{-}}p(x)=2\) Points on the left branch get close to the open point at height 2.左支上的點靠近 \(x=2\) 時會趨近空心點高度 2。
Right limit右極限 \(\lim_{x\to 2^{+}}p(x)=5\) Points on the right branch get close to the open point at height 5.右支上的點靠近 \(x=2\) 時會趨近空心點高度 5。
Two-sided雙邊 does not exist不存在 The two one-sided limits are different (2 versus 5).左右極限不同（2 與 5），所以雙邊極限不存在。
\(p(2)\) \(1\) The filled dot at \(x=2\) sits at height 1.\(x=2\) 的實心點在高度 1。

Rule of thumb · check both sides first口訣 · 先看兩側

At a possible jump, never read only one side. The two-sided limit can fail even when \(p(a)\) is defined.遇到可能跳躍時，不要只看單側。即使 \(p(a)\) 有定義，雙邊極限仍可能不存在。

3 · Exercises3 · 練習

Exercise 1 · Removable discontinuity練習 1 · 可去間斷

Use the graph of \(h\). It follows one line trend, has a hole at \((2,4)\), and has a filled point at \((2,2)\).請使用 \(h\) 的圖形。它沿著同一直線趨勢，在 \((2,4)\) 有空洞，並在 \((2,2)\) 有實心點。

Answer the following:回答下列問題：

  1. Find \(\lim_{x\to 2^{-}}h(x)\).求 \(\lim_{x\to 2^{-}}h(x)\)。
  2. Find \(\lim_{x\to 2^{+}}h(x)\).求 \(\lim_{x\to 2^{+}}h(x)\)。
  3. Does \(\lim_{x\to 2}h(x)\) exist? If yes, what is it?\(\lim_{x\to 2}h(x)\) 是否存在？若存在，值為何？
  4. Find \(h(2)\).求 \(h(2)\)。
  5. Is \(h\) continuous at \(x=2\)? Explain in one sentence.\(h\) 在 \(x=2\) 是否連續？請用一句話說明。
Hint:提示： The hole gives the trend value, and the filled dot gives the actual value at the point.空心點給的是趨勢值，實心點給的是該點實際函數值。
Show full solution顯示完整解答

(1) Along the line from the left, height approaches 4.(1) 由左側沿著線段靠近時，高度趨近 4。

\[ \lim_{x\to 2^{-}}h(x)=4. \]

(2) From the right, the same line also approaches 4.(2) 由右側靠近時，沿同一直線也趨近 4。

\[ \lim_{x\to 2^{+}}h(x)=4. \]

(3) Both one-sided limits are 4, so the two-sided limit exists.(3) 左右極限都為 4，所以雙邊極限存在。

\[ \lim_{x\to 2}h(x)=4. \]

(4) The filled point at \(x=2\) is at height 2.(4) \(x=2\) 的實心點在高度 2。

\[ h(2)=2. \]

(5) Not continuous at \(x=2\), because \(\lim_{x\to 2}h(x)=4\neq h(2)=2\).(5) 在 \(x=2\) 不連續，因為 \(\lim_{x\to 2}h(x)=4\neq h(2)=2\)。

Sense check. The trend value 4 and actual value 2 both lie in the visible range of the graph, so the reading is geometrically consistent.合理性檢查。趨勢值 4 與實際值 2 都落在圖上可見高度範圍內，判讀在幾何上是合理的。

Exercise 2 · Jump discontinuity練習 2 · 跳躍間斷

Use the graph of \(q\). The left branch approaches \((2,2)\) (open), and the right branch starts at \((2,5)\) (open). A filled point is plotted at \((2,3)\).請使用 \(q\) 的圖形。左支靠近 \((2,2)\)（空心），右支由 \((2,5)\)（空心）開始，並在 \((2,3)\) 畫有實心點。

Answer the following:回答下列問題：

  1. Find \(\lim_{x\to 2^{-}}q(x)\).求 \(\lim_{x\to 2^{-}}q(x)\)。
  2. Find \(\lim_{x\to 2^{+}}q(x)\).求 \(\lim_{x\to 2^{+}}q(x)\)。
  3. Does \(\lim_{x\to 2}q(x)\) exist? Explain.\(\lim_{x\to 2}q(x)\) 是否存在？請說明。
  4. Find \(q(2)\).求 \(q(2)\)。
  5. Can a function be defined at \(x=2\) even when the two-sided limit does not exist?即使雙邊極限不存在，函數是否仍可在 \(x=2\) 定義？
Hint:提示： Compare the two open-point heights at \(x=2\). Then read \(q(2)\) from the filled point.先比較 \(x=2\) 處兩個空心點的高度，再由實心點讀出 \(q(2)\)。
Show full solution顯示完整解答

(1) On the left branch, heights approach 2.(1) 在左支上，靠近時高度趨近 2。

\[ \lim_{x\to 2^{-}}q(x)=2. \]

(2) On the right branch, heights approach 5.(2) 在右支上，靠近時高度趨近 5。

\[ \lim_{x\to 2^{+}}q(x)=5. \]

(3) The two one-sided limits are different, so the two-sided limit does not exist.(3) 左右極限不同，所以雙邊極限不存在。

\[ \lim_{x\to 2}q(x)\text{ does not exist}. \]

(4) The filled point at \(x=2\) is at height 3.(4) \(x=2\) 的實心點在高度 3。

\[ q(2)=3. \]

(5) Yes. A function value can be assigned at \(x=2\) even if \(\lim_{x\to 2}q(x)\) does not exist.(5) 可以。即使 \(\lim_{x\to 2}q(x)\) 不存在，仍可在 \(x=2\) 指定函數值。

Sense check. The one-sided limits 2 and 5 and the point value 3 are all visible on the graph at the same input \(x=2\), which is exactly the signature of a jump with a separately defined point.合理性檢查。在同一輸入 \(x=2\) 上，左趨近值 2、右趨近值 5、點值 3 同時可見，這正是跳躍且另行定義點值的典型圖形。
