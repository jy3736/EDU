1.2 Computing Limits1.2 計算極限

1 · Concept and Fundamental Knowledge1 · 概念與基礎知識

Definition · Computing a limit定義 · 計算極限

To compute \(\lim_{x\to a}f(x)\), we identify the expression type and apply a valid rule so the nearby behavior can be read as a number. Sometimes direct substitution works immediately. Sometimes we must first simplify the expression (factor, cancel, or rewrite), and then substitute.要計算 \(\lim_{x\to a}f(x)\)，先判斷表達式型態，再使用正確規則把附近行為化為數值。有時可直接代入；有時需先化簡（因式分解、約分或改寫）後再代入。

Graphically, the limit is still read from where the curve goes near \(x=a\). Algebra gives a fast and reliable way to get the same value when graph reading is rough.在圖形上，極限仍是看曲線在 \(x=a\) 附近往哪裡走。代數方法則在圖形不易精讀時，提供快速且可靠的同一個答案。

Key terms · common algebra patterns重要名詞 · 常見代數型態

  • Direct substitution form. If plugging in \(x=a\) gives a regular number, that number is the limit.直接代入型。若代入 \(x=a\) 後得到一般數值，該數值就是極限。
  • Indeterminate form \(0/0\). This does not mean the limit is zero. It means we must simplify first.不定形 \(0/0\)。這不代表極限是 0，而是表示必須先化簡。
  • Removable hole. After cancellation, the simplified function gives the trend value at the hole.可去空洞。約分後的簡化函數會給出空洞處的趨勢值。
  • One-sided check. If left and right limits differ, the two-sided limit does not exist.單邊檢查。若左右極限不同，雙邊極限不存在。

Rule · Compute in this order規則 · 依序計算

Try direct substitution first. If you get \(0/0\), simplify algebraically and then substitute again. If the expression changes sign across \(a\), compute left and right limits separately before deciding the two-sided limit.先嘗試直接代入。若得到 \(0/0\)，先做代數化簡，再代入一次。若表達式在 \(a\) 左右會改變符號，先分別算左右極限，再判定雙邊極限。

2 · Worked Examples2 · 範例（詳解）

Example 1 · Direct substitution works範例 1 · 直接代入可行

For \(f(x)=2x+1\), compute \(\lim_{x\to 2}f(x)\). The graph is a straight line, so the trend value should match substitution.對 \(f(x)=2x+1\)，計算 \(\lim_{x\to 2}f(x)\)。圖形是一條直線，因此趨勢值應與代入值一致。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
Expression表達式 \(f(x)=2x+1\) A linear function, continuous everywhere.一次函數，在每一點都連續。
Substitute代入 \(2(2)+1=5\) Direct substitution gives an ordinary number.直接代入得到一般數值。
Limit極限 \(\lim_{x\to 2}(2x+1)=5\) Graph and algebra agree at height 5.圖形與代數都得到高度 5。

The first move in limit computation is simple: try substitution first.計算極限的第一步很直接：先試代入。

Example 2 · Factor and cancel a \(0/0\)範例 2 · \(0/0\) 先因式分解再約分

Compute \(\lim_{x\to 2}\dfrac{x^2-4}{x-2}\). The graph is a line with a hole at \((2,4)\), so the trend should be 4.計算 \(\lim_{x\to 2}\dfrac{x^2-4}{x-2}\)。圖形是直線但在 \((2,4)\) 有一個空洞，所以趨勢值應是 4。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
Direct form直接型態 \(\dfrac{0}{0}\) At \(x=2\): numerator \(=0\), denominator \(=0\), so simplify first.在 \(x=2\) 時：分子 \(=0\)、分母 \(=0\)，所以先化簡。
Simplify化簡 \(\dfrac{(x-2)(x+2)}{x-2}=x+2\) For \(x\ne 2\), cancel \((x-2)\).當 \(x\ne 2\) 時可約分 \((x-2)\)。
Limit極限 \(\lim_{x\to 2}\dfrac{x^2-4}{x-2}=4\) Use simplified form \(x+2\), then substitute \(x=2\).使用簡化後 \(x+2\)，再代入 \(x=2\)。

Notice · undefined point, defined trend注意 · 點值未定義，趨勢仍可定義

The function is not defined at \(x=2\), but the limit still exists because nearby values approach 4 from both sides.函數在 \(x=2\) 雖未定義，但因兩側附近值都趨近 4，極限仍存在。

Example 3 · One-sided limits disagree範例 3 · 左右極限不一致

Compute \(\lim_{x\to 2}\left(\dfrac{|x-2|}{x-2}+3\right)\). This expression gives one value on the left and another on the right.計算 \(\lim_{x\to 2}\left(\dfrac{|x-2|}{x-2}+3\right)\)。此表達式在左側與右側會給出不同值。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
Left limit左極限 \(\lim_{x\to 2^{-}}\left(\dfrac{|x-2|}{x-2}+3\right)=2\) For \(x\lt 2\), \(|x-2|=-(x-2)\), so value is \(-1+3=2\).當 \(x\lt 2\) 時，\(|x-2|=-(x-2)\)，故值為 \(-1+3=2\)。
Right limit右極限 \(\lim_{x\to 2^{+}}\left(\dfrac{|x-2|}{x-2}+3\right)=4\) For \(x\gt 2\), \(|x-2|=x-2\), so value is \(1+3=4\).當 \(x\gt 2\) 時，\(|x-2|=x-2\)，故值為 \(1+3=4\)。
Two-sided雙邊 does not exist不存在 Left and right limits are different, so no two-sided limit.左右極限不同，因此雙邊極限不存在。

Rule of thumb · absolute value needs side analysis口訣 · 絕對值要分左右

Near the switching point of an absolute value, always compute left and right expressions separately.在絕對值切換點附近，務必把左側與右側表達式分開計算。

3 · Exercises3 · 練習

Exercise 1 · Factor then compute練習 1 · 先分解後計算

Use the graph of \(m(x)=\dfrac{x^2-9}{x-3}\) (\(x\ne 3\)). The graph follows a line with a hole at \((3,6)\).使用 \(m(x)=\dfrac{x^2-9}{x-3}\)（\(x\ne 3\)）的圖形。此圖沿著一直線，但在 \((3,6)\) 有空洞。

Answer all five parts:回答下列五題：

  1. Evaluate \(\lim_{x\to 3^{-}}m(x)\).求 \(\lim_{x\to 3^{-}}m(x)\)。
  2. Evaluate \(\lim_{x\to 3^{+}}m(x)\).求 \(\lim_{x\to 3^{+}}m(x)\)。
  3. Does \(\lim_{x\to 3}m(x)\) exist, and what is it?\(\lim_{x\to 3}m(x)\) 是否存在？值為何？
  4. Is \(m(3)\) defined?\(m(3)\) 是否有定義？
  5. Use factoring to confirm your limit value.用因式分解確認你的極限值。
Hint:提示： Write \(x^2-9=(x-3)(x+3)\). The simplified trend expression is \(x+3\) for \(x\ne 3\).將 \(x^2-9\) 寫成 \((x-3)(x+3)\)。對 \(x\ne 3\) 而言，簡化後的趨勢式是 \(x+3\)。
Show full solution顯示完整解答

(1) From the left branch, heights approach 6.(1) 由左側分支靠近時，高度趨近 6。

\[ \lim_{x\to 3^{-}}m(x)=6. \]

(2) From the right branch, heights also approach 6.(2) 由右側分支靠近時，高度也趨近 6。

\[ \lim_{x\to 3^{+}}m(x)=6. \]

(3) Left and right agree, so the two-sided limit exists and equals 6.(3) 左右一致，因此雙邊極限存在且等於 6。

\[ \lim_{x\to 3}m(x)=6. \]

(4) \(m(3)\) is undefined in the original formula because denominator \(x-3=0\).(4) 原式在 \(x=3\) 時分母 \(x-3=0\)，故 \(m(3)\) 無定義。

\[ m(3)\text{ is undefined}. \]

(5) Algebra check:(5) 代數確認：

\[ \frac{x^2-9}{x-3}=\frac{(x-3)(x+3)}{x-3}=x+3\quad (x\ne 3). \]

\[ \lim_{x\to 3}m(x)=\lim_{x\to 3}(x+3)=6. \]

Sense check. The graph has a hole at \((3,6)\), so a defined trend to 6 with an undefined point value is exactly what the picture shows.合理性檢查。圖上在 \((3,6)\) 有空洞，因此「趨勢為 6、點值未定義」正是圖形所呈現的結果。

Exercise 2 · Two-sided limit test by sides練習 2 · 以左右判定雙邊極限

Use the graph of \(q(x)=\dfrac{|x-2|}{x-2}+3\) for \(x\ne 2\). A separate filled point at \((2,3)\) is also shown.使用 \(q(x)=\dfrac{|x-2|}{x-2}+3\)（\(x\ne 2\)）的圖形。圖上另外給出實心點 \((2,3)\)。

Answer all five parts:回答下列五題：

  1. Compute \(\lim_{x\to 2^{-}}q(x)\).計算 \(\lim_{x\to 2^{-}}q(x)\)。
  2. Compute \(\lim_{x\to 2^{+}}q(x)\).計算 \(\lim_{x\to 2^{+}}q(x)\)。
  3. Does \(\lim_{x\to 2}q(x)\) exist?\(\lim_{x\to 2}q(x)\) 是否存在？
  4. Read the plotted value at \(x=2\).讀出 \(x=2\) 的作圖值。
  5. State in one sentence why one-sided analysis is required here.用一句話說明此題為何必須做單邊分析。
Hint:提示： For \(x\lt 2\), \(\dfrac{|x-2|}{x-2}=-1\); for \(x\gt 2\), \(\dfrac{|x-2|}{x-2}=1\).當 \(x\lt 2\) 時，\(\dfrac{|x-2|}{x-2}=-1\)；當 \(x\gt 2\) 時，\(\dfrac{|x-2|}{x-2}=1\)。
Show full solution顯示完整解答

(1) Left side gives \(-1+3=2\).(1) 左側代入得到 \(-1+3=2\)。

\[ \lim_{x\to 2^{-}}q(x)=2. \]

(2) Right side gives \(1+3=4\).(2) 右側代入得到 \(1+3=4\)。

\[ \lim_{x\to 2^{+}}q(x)=4. \]

(3) Because \(2\ne 4\), the two-sided limit does not exist.(3) 因為 \(2\ne 4\)，雙邊極限不存在。

\[ \lim_{x\to 2}q(x)\text{ does not exist}. \]

(4) The filled point at \(x=2\) is at height 3.(4) \(x=2\) 的實心點位於高度 3。

\[ q(2)=3. \]

(5) One-sided analysis is required because the absolute value changes formula at \(x=2\), producing different left and right trends.(5) 必須做單邊分析，因為絕對值在 \(x=2\) 會切換公式，導致左右趨勢不同。

Sense check. The graph clearly shows approach to 2 from the left and 4 from the right, with a separate point value 3 at \(x=2\), so every computed value matches the picture.合理性檢查。圖形清楚顯示左趨近 2、右趨近 4，且在 \(x=2\) 另有點值 3，因此計算結果與圖形完全一致。
