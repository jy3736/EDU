# Probability and Statistics for Electronic Engineering
## Learning-by-Doing Digital Teaching Material Development Master Specification

Version: 1.0
Institution: Southern Taiwan University of Science and Technology (STUST)
Department: Electronic Engineering
Target Course: Probability and Statistics (36D00301)

## Project Purpose

This repository serves as the complete digital teaching-material ecosystem for Probability and Statistics within the Electronic Engineering curriculum.

Instructional philosophy:

Observation → Prediction → Experiment → Analysis → Theory → Application

## Assumed Course Scope

1. Probability Models
2. Set Theory
3. Counting Techniques
4. Probability Axioms
5. Conditional Probability
6. Independence
7. Discrete Random Variables
8. Continuous Random Variables
9. Expectation
10. Variance
11. Cumulative Distribution Functions
12. Joint Distributions
13. Random Vectors
14. Engineering Applications
15. MATLAB-Based Simulations

## Educational Philosophy

### Learning-by-Doing

Every major concept must include:

- Prediction
- Experiment
- Observation
- Interpretation

### Engineering Context First

Every module must answer:

Why does an electronic engineer need this?

### Review-Flipped Learning

Supports:

- Before-class review
- During-class participation
- After-class reinforcement

### AI-Augmented Learning

Students may use AI for:

- Concept explanation
- Additional examples
- Code understanding
- Reflection activities
