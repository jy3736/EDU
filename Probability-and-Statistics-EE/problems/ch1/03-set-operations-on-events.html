1.3 Set Operations on Events1.3 事件的集合運算

1 · Concept and Fundamental Knowledge1 · 概念與基礎知識

Why an electronic engineer needs this. Engineering questions are built from “and,” “or,” and “not.” A backup link works if the main link or the spare works; a board passes if it fails neither test; an alarm fires if temperature and current are both high. Each phrase is a set operation on events, and each has a probability you can compute from a Venn diagram.電子工程師為什麼需要這個。工程問題常由「且」「或」「非」組成。主鏈路或備援鏈路只要有一條可用，備援路徑就可用；一片板子在兩項測試都沒有失敗時通過；溫度且電流都偏高時警報會觸發。每個語句都是事件上的集合運算，也都有可以從范氏圖算出的機率。

Definition · union, intersection, complement, difference定義 · 聯集、交集、補集、差集

For events \(A\) and \(B\) in a sample space \(S\): the union \(A\cup B\) is “\(A\) or \(B\) (or both),” the intersection \(A\cap B\) is “\(A\) and \(B\),” the complement \(\overline{A}\) is “not \(A\),” and the difference \(A\setminus B\) is “\(A\) but not \(B\).” The “or” here is always inclusive: \(A\cup B\) includes the overlap.對樣本空間 \(S\) 中的事件 \(A\) 與 \(B\)：聯集 \(A\cup B\) 是「\(A\) 或 \(B\)（也可以兩者都發生）」，交集 \(A\cap B\) 是「\(A\) 且 \(B\)」，補集 \(\overline{A}\) 是「非 \(A\)」，差集 \(A\setminus B\) 是「\(A\) 但非 \(B\)」。這裡的「或」一律是相容或：\(A\cup B\) 會包含重疊區。

The shaded region below is the union \(A\cup B\): every outcome in either circle, with the overlap included once. Operations are just different regions you shade on the same diagram.下方著色區域是聯集 \(A\cup B\)：屬於任一圓的所有結果，重疊區只算一次。各種運算其實就是在同一張圖上著色不同區域。

Rule · the addition (inclusion-exclusion) law規則 · 加法（排容）律

If you add \(P(A)\) and \(P(B)\) you count the overlap twice, so subtract it once:如果直接把 \(P(A)\) 與 \(P(B)\) 相加，重疊區會被算兩次，所以要扣回一次：

\[ P(A\cup B)=P(A)+P(B)-P(A\cap B). \]

When \(A\) and \(B\) are mutually exclusive, \(P(A\cap B)=0\) and this collapses to \(P(A\cup B)=P(A)+P(B)\).當 \(A\) 與 \(B\) 互斥時，\(P(A\cap B)=0\)，此式化為 \(P(A\cup B)=P(A)+P(B)\)。

Key terms · complement and De Morgan重要名詞 · 補集與笛摩根

  • Complement rule. \(P(\overline{A})=1-P(A)\). Often the fastest route: “at least one” is usually easier as “\(1-\) none.”補集律。\(P(\overline{A})=1-P(A)\)。這常是最快的算法：「至少一個」通常改成「\(1-\) 一個都沒有」會更好算。
  • De Morgan laws. “Not (\(A\) or \(B\))” equals “(not \(A\)) and (not \(B\)),” and “not (\(A\) and \(B\))” equals “(not \(A\)) or (not \(B\))”: \(\overline{A\cup B}=\overline{A}\cap\overline{B}\) and \(\overline{A\cap B}=\overline{A}\cup\overline{B}\).笛摩根定律。「非（\(A\) 或 \(B\)）」等於「（非 \(A\)）且（非 \(B\)）」，「非（\(A\) 且 \(B\)）」等於「（非 \(A\)）或（非 \(B\)）」：\(\overline{A\cup B}=\overline{A}\cap\overline{B}\) 且 \(\overline{A\cap B}=\overline{A}\cup\overline{B}\)。
  • Difference. \(A\setminus B\) is the “only \(A\)” region, with probability \(P(A)-P(A\cap B)\).差集。\(A\setminus B\) 是「僅 \(A\)」區域，機率為 \(P(A)-P(A\cap B)\)。

Simulate it first先模擬看看

Predict: with region probabilities only-\(A\) \(0.30\), both \(0.20\), only-\(B\) \(0.20\), neither \(0.30\), what is \(P(A\cup B)\)? Guess, then check that the simulation agrees with the addition law.預測：區域機率為僅 \(A\) \(0.30\)、兩者 \(0.20\)、僅 \(B\) \(0.20\)、皆非 \(0.30\)，那麼 \(P(A\cup B)\) 是多少？先猜，再確認模擬與加法律一致。

import numpy as np
rng = np.random.default_rng(0)

regions = ["onlyA", "both", "onlyB", "neither"]
probs   = [0.30,    0.20,   0.20,    0.30]
draws = rng.choice(regions, size=200_000, p=probs)

A = np.isin(draws, ["onlyA", "both"])
B = np.isin(draws, ["onlyB", "both"])
union = A | B            # A or B
inter = A & B            # A and B
print(union.mean())                                  # estimate of P(A or B)
print(A.mean() + B.mean() - inter.mean())            # addition law, same value

Observed: both print about \(0.700\), matching \(P(A\cup B)=0.50+0.40-0.20=0.70\).觀察結果：兩者都印出約 \(0.700\)，對應 \(P(A\cup B)=0.50+0.40-0.20=0.70\)。

2 · Worked Examples2 · 範例（詳解）

Example 1 · Union by the addition law範例 1 · 以加法律求聯集

Two events satisfy \(P(A)=0.50\), \(P(B)=0.40\), and \(P(A\cap B)=0.20\). The shaded region \(A\cup B\) is below. Find \(P(A\cup B)\) and the difference \(P(A\setminus B)\).兩事件滿足 \(P(A)=0.50\)、\(P(B)=0.40\)、\(P(A\cap B)=0.20\)。下方為著色的 \(A\cup B\)。求 \(P(A\cup B)\) 與差集 \(P(A\setminus B)\)。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
\(P(A\cup B)\) \(0.70\) Addition law: \(0.50+0.40-0.20\); the overlap is removed so it is not double counted.加法律：\(0.50+0.40-0.20\)；扣除重疊以免重複計算。
\(P(A\setminus B)\) \(0.30\) “Only \(A\)” is \(P(A)-P(A\cap B)=0.50-0.20\).「僅 \(A\)」為 \(P(A)-P(A\cap B)=0.50-0.20\)。
\(P(\overline{A\cup B})\) \(0.30\) Complement of the shaded region: \(1-0.70\).著色區的補集：\(1-0.70\)。

The one move that matters is subtracting the overlap exactly once. Skipping it would give \(0.90\), which over-counts the \(0.20\) shared by both events.關鍵動作是把重疊恰好扣一次。略過它會得到 \(0.90\)，這多算了兩事件共用的 \(0.20\)。

Example 2 · “Neither” and De Morgan範例 2 · 「皆非」與笛摩根

Using the same numbers (\(P(A)=0.50\), \(P(B)=0.40\), \(P(A\cap B)=0.20\)), find the probability that neither event happens. The shaded region below is everything outside both circles. The new wrinkle is recognising it two ways, which is De Morgan’s law in action.沿用相同數據（\(P(A)=0.50\)、\(P(B)=0.40\)、\(P(A\cap B)=0.20\)），求兩者都不發生的機率。下方著色區是兩圓之外的全部。這題的新重點，是用兩種方式認出同一塊區域，也就是笛摩根定律的用法。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
\(P(A\cup B)\) \(0.70\) From Example 1, the union.承範例 1 的聯集。
\(P(\overline{A\cup B})\) \(0.30\) Complement rule: \(1-P(A\cup B)=1-0.70\).補集律：\(1-P(A\cup B)=1-0.70\)。
\(P(\overline{A}\cap\overline{B})\) \(0.30\) De Morgan: “neither” is the same shaded region, so the value is identical.笛摩根：「皆非」就是同一塊著色區，所以數值相同。

Notice · two names, one region注意 · 兩種名稱，同一區域

\(\overline{A\cup B}\) (“not (\(A\) or \(B\))”) and \(\overline{A}\cap\overline{B}\) (“(not \(A\)) and (not \(B\))”) describe the very same shaded area. That equality is De Morgan’s law; the diagram makes it obvious rather than something to memorise.\(\overline{A\cup B}\)（「非（\(A\) 或 \(B\)）」）與 \(\overline{A}\cap\overline{B}\)（「（非 \(A\)）且（非 \(B\)）」）描述的是同一塊著色區。這個相等就是笛摩根定律；用圖看會很直覺，不需要硬背。

Example 3 · A redundant link (engineering)範例 3 · 備援鏈路（工程應用）

A data path has a main link and a spare. Let \(A=\{\text{main works}\}\) and \(B=\{\text{spare works}\}\), with \(P(A)=0.90\), \(P(B)=0.80\), and (from field data) \(P(A\cap B)=0.74\). The path carries data if either link works, the event \(A\cup B\). Find the probability the path works, and the probability it fails.一條資料路徑有主鏈路與備援鏈路。令 \(A=\{\text{主鏈路可用}\}\)、\(B=\{\text{備援可用}\}\)，且 \(P(A)=0.90\)、\(P(B)=0.80\)，（依現場資料）\(P(A\cap B)=0.74\)。只要任一鏈路可用，路徑即可傳資料，即事件 \(A\cup B\)。求路徑可用的機率，以及失效的機率。

Reading it off, step by step逐步判讀

Quantity量 Value值 How you read it判讀方式
\(P(A\cup B)\) \(0.96\) Addition law: \(0.90+0.80-0.74\). The path works.加法律：\(0.90+0.80-0.74\)。路徑可用。
\(P(\text{fails})\) \(0.04\) Path fails only if both links fail: \(1-P(A\cup B)=\overline{A}\cap\overline{B}\).路徑僅在兩條鏈路皆失效時才失效：\(1-P(A\cup B)=\overline{A}\cap\overline{B}\)。

Redundancy pays off: each link alone fails \(10\%\) or \(20\%\) of the time, but the pair fails only \(4\%\) of the time. Note we used the measured \(P(A\cap B)=0.74\) directly; we did not assume the links fail independently, a separate idea coming in Chapter 3.備援確實有效：單一鏈路各有 \(10\%\) 或 \(20\%\) 的失效率，但兩條一起看時只剩 \(4\%\) 失效。注意這裡直接使用量測值 \(P(A\cap B)=0.74\)；我們沒有假設兩條鏈路獨立失效，獨立性會在第 3 章另外討論。

3 · Exercises3 · 練習

Exercise 1 · Shading the operations練習 1 · 為運算著色

The Venn diagram gives the four region probabilities for events \(A\) and \(B\). Use them for every part.下方范氏圖給出事件 \(A\) 與 \(B\) 的四個區域機率。各小題均使用之。

Answer all five parts:回答以下五個小題：

  1. Find \(P(A)\) and \(P(B)\).求 \(P(A)\) 與 \(P(B)\)。
  2. Find \(P(A\cup B)\) using the addition law, and check it against the regions.以加法律求 \(P(A\cup B)\)，並用各區域核對。
  3. Find \(P(A\setminus B)\), the probability of “\(A\) but not \(B\).”求 \(P(A\setminus B)\)，即「\(A\) 但非 \(B\)」的機率。
  4. Find \(P(\overline{A\cup B})\).求 \(P(\overline{A\cup B})\)。
  5. Confirm De Morgan numerically: check \(P(\overline{A}\cap\overline{B})\) equals your answer to part 4.用數值確認笛摩根定律：檢查 \(P(\overline{A}\cap\overline{B})\) 是否等於第 4 小題的答案。
Hint:提示： Adding the three shaded regions that make up \(A\cup B\) should give the same number as the addition law. The “neither” region is the complement of that.把組成 \(A\cup B\) 的三塊著色區相加，應與加法律得到相同數值。「皆非」區即其補集。
Show full solution顯示完整解答

(1) Single events. Each circle is its two regions:(1) 單一事件。每個圓為其兩塊區域：

\[ P(A)=0.35+0.15=0.50,\qquad P(B)=0.25+0.15=0.40. \]

(2) Union. Addition law, then the region check:(2) 聯集。加法律，再以區域核對：

\[ P(A\cup B)=0.50+0.40-0.15=0.75=\underbrace{0.35+0.15+0.25}_{\text{three regions}}. \]

(3) Difference. “Only \(A\)” is the left crescent:(3) 差集。「僅 \(A\)」為左側月牙：

\[ P(A\setminus B)=P(A)-P(A\cap B)=0.50-0.15=0.35. \]

(4) Complement of the union.(4) 聯集的補集。

\[ P(\overline{A\cup B})=1-0.75=0.25. \]

(5) De Morgan check. “Neither” is the region outside both circles, read directly as \(0.25\), which equals \(P(\overline{A}\cap\overline{B})\). It matches part 4, confirming \(\overline{A\cup B}=\overline{A}\cap\overline{B}\).(5) 笛摩根驗證。「皆非」是兩圓之外的區域，直接讀得 \(0.25\)，即 \(P(\overline{A}\cap\overline{B})\)。與第 4 小題相符，確認 \(\overline{A\cup B}=\overline{A}\cap\overline{B}\)。

Sense check. The shaded union \(0.75\) and the “neither” \(0.25\) add to \(1\), as a region and its complement must.合理性檢查。著色聯集 \(0.75\) 與「皆非」\(0.25\) 相加為 \(1\)，符合區域與其補集之關係。

Exercise 2 · Working back to the overlap練習 2 · 反推重疊

A production line is monitored for two alarms. From the logs, \(P(A)=0.60\), \(P(B)=0.50\), and \(P(A\cup B)=0.80\). The overlap is not given; the diagram leaves it as a question mark for you to find.一條產線監看兩種警報。由紀錄可知 \(P(A)=0.60\)、\(P(B)=0.50\)、\(P(A\cup B)=0.80\)。重疊沒有直接給出；圖中以問號標示，要由你求出。

Answer all five parts:回答以下五個小題：

  1. Rearrange the addition law to find \(P(A\cap B)\).重排加法律以求 \(P(A\cap B)\)。
  2. Find \(P(A\setminus B)\) and \(P(B\setminus A)\).求 \(P(A\setminus B)\) 與 \(P(B\setminus A)\)。
  3. Find \(P(\overline{A\cup B})\), the probability that no alarm fires.求 \(P(\overline{A\cup B})\)，即沒有警報觸發的機率。
  4. Find \(P(\overline{A\cap B})\), the probability that not both alarms fire, and confirm it equals \(\,P(\overline{A}\cup\overline{B})\,\) (De Morgan).求 \(P(\overline{A\cap B})\)，即非兩警報同時觸發的機率，並確認其等於 \(\,P(\overline{A}\cup\overline{B})\,\)（笛摩根）。
  5. Could \(P(A\cup B)\) have been \(0.40\) with these \(P(A)\) and \(P(B)\)? Explain.在這組 \(P(A)\) 與 \(P(B)\) 下，\(P(A\cup B)\) 有可能是 \(0.40\) 嗎？請說明。
Hint:提示： Solve \(P(A\cup B)=P(A)+P(B)-P(A\cap B)\) for the overlap. For part 5, recall that \(A\cup B\) can never be smaller than \(A\) alone.由 \(P(A\cup B)=P(A)+P(B)-P(A\cap B)\) 解出重疊。第 5 小題請記得 \(A\cup B\) 永遠不可能小於單獨的 \(A\)。
Show full solution顯示完整解答

(1) The overlap. Rearranging the addition law:(1) 重疊。重排加法律：

\[ P(A\cap B)=P(A)+P(B)-P(A\cup B)=0.60+0.50-0.80=0.30. \]

(2) The two differences.(2) 兩個差集。

\[ P(A\setminus B)=0.60-0.30=0.30,\qquad P(B\setminus A)=0.50-0.30=0.20. \]

(3) No alarm.(3) 無警報。

\[ P(\overline{A\cup B})=1-0.80=0.20. \]

(4) Not both. Use the complement rule, then De Morgan:(4) 不是兩者都發生。用補集律，再用笛摩根：

\[ P(\overline{A\cap B})=1-P(A\cap B)=1-0.30=0.70=P(\overline{A}\cup\overline{B}). \]

(5) Could it be \(0.40\)? No. Since \(A\) is part of \(A\cup B\), we must have \(P(A\cup B)\ge P(A)=0.60\). A union of \(0.40\) would be smaller than \(A\) alone, which is impossible. (Equivalently, it would force \(P(A\cap B)=0.70\), larger than \(P(B)=0.50\).)(5) 有可能是 \(0.40\) 嗎？否。由於 \(A\) 是 \(A\cup B\) 的一部分，必有 \(P(A\cup B)\ge P(A)=0.60\)。聯集為 \(0.40\) 會小於單獨的 \(A\)，不可能。（等價地，這會迫使 \(P(A\cap B)=0.70\)，大於 \(P(B)=0.50\)。）

Sense check. The four regions are only-\(A\) \(0.30\), both \(0.30\), only-\(B\) \(0.20\), neither \(0.20\); they sum to \(1\), and the overlap \(0.30\) does not exceed either \(P(A)\) or \(P(B)\). Consistent.合理性檢查。四個區域為僅 \(A\) \(0.30\)、兩者 \(0.30\)、僅 \(B\) \(0.20\)、皆非 \(0.20\)；總和為 \(1\)，且重疊 \(0.30\) 未超過 \(P(A)\) 或 \(P(B)\)。一致。
