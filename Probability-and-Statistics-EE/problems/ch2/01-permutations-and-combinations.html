2.1 Permutations and Combinations2.1 排列與組合

1 · Concept and Fundamental Knowledge1 · 概念與基礎知識

Why an electronic engineer needs this. Before you can compute many probabilities, you must count how many bit patterns, passwords, test orders, or selected sensors are possible. Counting is the bridge between a design space and a probability model.電子工程師為什麼需要這個。很多機率還沒開始算之前，得先數清楚有多少位元樣式、密碼、測試順序或感測器選取方式。計數是設計空間和機率模型之間的橋樑。

Definition · permutation and combination定義 · 排列與組合

A permutation counts ordered selections: changing the order creates a different outcome. A combination counts unordered selections: changing the order does not create a new outcome.排列是在數「有順序」的選取方式：順序一變，就算不同結果。組合是在數「無順序」的選取方式：順序改變，不會產生新結果。

Key terms · the counting language重要名詞 · 計數語言

  • Factorial. \(n!\) is the number of ways to arrange \(n\) distinct objects: \(n!=n(n-1)\cdots 2\cdot 1\).階乘。\(n!\) 是排列 \(n\) 個不同物件的方法數：\(n!=n(n-1)\cdots 2\cdot 1\)。
  • Permutation. \(P(n,k)\) counts ordered selections of \(k\) objects from \(n\).排列。\(P(n,k)\) 用來計算從 \(n\) 個物件中選 \(k\) 個、且順序重要的方法數。
  • Combination. \(\binom{n}{k}\) counts unordered selections of \(k\) objects from \(n\).組合。\(\binom{n}{k}\) 計算從 \(n\) 個物件中選 \(k\) 個且順序不重要的方法數。

Simulate it first先模擬看看

Predict: if a 4-digit PIN is chosen uniformly from 0000 to 9999, about what fraction of random guesses hit the target PIN 0427?預測：如果四位數 PIN 在 0000 到 9999 之間等可能產生，隨機猜一次命中目標 PIN 0427 的比例大約是多少？

import numpy as np
rng = np.random.default_rng(0)

guesses = rng.integers(0, 10_000, size=500_000)
hits = (guesses == 427)
print(hits.mean())

Observed: about \(0.0001\). There are \(10^4\) ordered digit strings, so one target has probability \(1/10000\).觀察結果：約 \(0.0001\)。四位數 PIN 共有 \(10^4\) 種有順序的可能，因此單一目標的機率為 \(1/10000\)。

Rule · choose the count by asking about order規則 · 先問順序重不重要

\[ P(n,k)=\frac{n!}{(n-k)!},\qquad \binom{n}{k}=\frac{n!}{k!(n-k)!}. \]

Use \(P(n,k)\) when slot 1, slot 2, and slot 3 have different meanings. Use \(\binom{n}{k}\) when the selected group is the only thing that matters.當第 1 格、第 2 格與第 3 格的意義不同時，用 \(P(n,k)\)。如果只在乎被選到的是哪一組，用 \(\binom{n}{k}\)。

2 · Worked Examples2 · 範例（詳解）

Example 1 · Three-bit words範例 1 · 三位元字組

How many 3-bit words can a register store?一個三位元暫存器可以存幾種不同字組？

Reading it off, step by step逐步判讀

Quantity量Value值How you read it判讀方式
Slots位置\(3\)The register has three ordered bit positions.三個位元位置是有順序的。
Choices選擇\(2,2,2\)Each bit is either 0 or 1.每個位元可為 0 或 1。
Total總數\(2^3=8\)Multiply the choices across the ordered slots.把每個有順序位置的選擇數相乘。

This is the multiplication principle: independent slot choices multiply.這就是乘法原理：各個位置的選擇數相乘。

Example 2 · A PIN with no repeated digit範例 2 · 不重複數字的 PIN

How many 4-digit PINs can be made from digits 0 to 9 if no digit repeats?如果數字不可重複，0 到 9 可以組成多少個四位數 PIN？

Reading it off, step by step逐步判讀

Quantity量Value值How you read it判讀方式
Order順序matters重要PIN 1234 and 4321 are different.PIN 1234 與 4321 不同。
Choices選擇\(10\cdot 9\cdot 8\cdot 7\)After each slot, one digit is no longer available.每填一格，就少一個可用數字。
Total總數\(P(10,4)=5040\)This is a permutation of 4 digits from 10.這是從 10 個數字中取 4 個、且順序重要的排列。

The new wrinkle is no repetition: the number of choices shrinks after each slot.這題多了「不可重複」這個條件：每填完一格，下一格可選的數字就少一個。

Example 3 · Choosing test sensors (engineering)範例 3 · 選取測試感測器（工程應用）

A test bench has 8 sensors, and the engineer will choose 3 to monitor during a quick diagnostic. How many sensor sets are possible?測試台有 8 個感測器，工程師要選 3 個做快速診斷。可能的感測器組合有幾種？

Reading it off, step by step逐步判讀

Quantity量Value值How you read it判讀方式
\(n\)\(8\)There are 8 available sensors.共有 8 個可用感測器。
\(k\)\(3\)The engineer chooses 3 sensors.工程師選取 3 個感測器。
Total總數\(\binom{8}{3}=56\)Order does not matter, so divide out the \(3!\) orders of each set.順序不重要，因此扣除每組的 \(3!\) 種排列。

The engineering clue is the word set: a sensor set is unchanged if you name its members in a different order.工程情境中的關鍵字是「組」：同一組感測器不會因列舉順序不同而改變。

3 · Exercises3 · 練習

Exercise 1 · Four test orders練習 1 · 四個測試順序

A technician will run 4 different tests chosen from 6 available tests. The order of the tests is recorded.技術員要從 6 個可用測試中挑 4 個不同測試來執行，而且會記錄執行順序。

Answer all five parts:回答以下五個小題：

  1. Does order matter?順序重不重要？
  2. May a test repeat?測試可不可以重複？
  3. Write the product of choices.寫出選擇數的乘積。
  4. Write the answer using \(P(n,k)\).用 \(P(n,k)\) 寫出答案。
  5. Compute the total number of orders.算出總共有幾種順序。
Hint:提示： The phrase "order is recorded" means this is a permutation.「會記錄順序」表示這是排列。
Show full solution顯示完整解答

(1) Order matters because first test and second test are different positions.(1) 順序重要，因為第一個測試與第二個測試是不同位置。

(2) No repeat, because the tests are different.(2) 不可以重複，因為題目說要選 4 個不同測試。

\[ 6\cdot 5\cdot 4\cdot 3. \]

\[ P(6,4)=\frac{6!}{(6-4)!}. \]

\[ P(6,4)=6\cdot 5\cdot 4\cdot 3=360. \]

Sense check. The answer is less than \(6^4=1296\), which would allow repeated tests, so 360 is reasonable.合理性檢查。答案小於允許重複的 \(6^4=1296\)，所以 360 合理。

Exercise 2 · Spare channel selection練習 2 · 備援通道選取

A board has 7 possible spare channels, and the designer chooses 2 to enable. The enabled set is all that matters.一片板子有 7 個可用的備援通道，設計者要啟用其中 2 個。這裡只在乎啟用的是哪兩個通道。

Answer all five parts:回答以下五個小題：

  1. Does order matter?順序重不重要？
  2. Which formula should you use?應該用哪個公式？
  3. Compute the number of enabled sets.算出可啟用的通道組合數。
  4. Explain why \(P(7,2)\) would over-count.說明為什麼 \(P(7,2)\) 會重複計數。
  5. How many ordered listings correspond to each unordered set?每個無順序組合會對應幾種有順序的列法？
Hint:提示： The set {channel 2, channel 5} is the same as {channel 5, channel 2}.集合 {通道 2, 通道 5} 與 {通道 5, 通道 2} 是同一組。
Show full solution顯示完整解答

(1) Order does not matter.(1) 順序不重要。

\[ \binom{7}{2}=\frac{7!}{2!5!}. \]

\[ \binom{7}{2}=21. \]

(4) \(P(7,2)=42\) counts channel 2 then 5 and channel 5 then 2 separately, so it doubles every set.(4) \(P(7,2)=42\) 會把先通道 2 後通道 5，以及先通道 5 後通道 2 分開計算，因此每組都算兩次。

\[ 2!=2. \]

Sense check. There are 21 choices, which is much smaller than \(2^7\) because the designer enables exactly two channels, not any number of channels.合理性檢查。共有 21 種，遠小於 \(2^7\)，因為設計者恰好啟用兩個通道，而不是任意數量。
