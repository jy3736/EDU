# EDU — Teaching Materials

Open, self-contained course study pages by [jy3736](https://github.com/jy3736),
published with GitHub Pages at <https://jy3736.github.io/EDU/>.

## Pages

- **[Calculus 101 — Introduction to Limits](https://jy3736.github.io/EDU/Calculus-101/)**
  — interactive limit exercises with custom SVG graphs and KaTeX-typeset math
  (removable discontinuity, one-sided limits, infinite limits, the squeeze
  theorem, continuity). Works offline; supports light/dark themes and EN / 中文.

## Notes

- Math is rendered with a locally vendored copy of [KaTeX](https://katex.org/)
  (MIT licensed) under `Calculus-101/vendor/katex/`.
- Calculus content is adapted from
  [OpenStax *Calculus Volume 1*](https://openstax.org/details/books/calculus-volume-1),
  licensed CC BY-NC-SA 4.0.
