# AI 代理工作坊 / AI Agent Workshop

用一份真實的 SIMS 學習動機問卷，帶大專教師練習指揮 AI 代理（Claude Code、Codex、Copilot）完成資料清理、信度分析、t 檢定與畫圖。
A hands-on workshop where college instructors drive an AI agent through data cleaning, reliability analysis, t-tests, and charts using a real SIMS learning-motivation survey.

## 選擇語言 / Choose a language

|---|---|

Both versions cover the same exercises, only the language differs. All outputs go to the shared [_results/](_results/) folder.
