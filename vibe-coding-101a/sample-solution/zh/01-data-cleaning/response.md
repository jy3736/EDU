# 練習 01 範例回覆 - 資料清理

> 產出的檔案在 [outputs/](outputs/)。

我打開那個工作表時，跳過了第一列的大標題、改用第二列當欄位名稱，刪掉全空白列，保留 `帳號`，把 16 題依序改名成 Q1 到 Q16，並把選項編號轉成實際同意分數：`actual_score = 8 - option_number`。

**結果**

- `clean_pre.csv`：**43 列**；`clean_post.csv`：**44 列**（不含表頭），欄位皆為 `帳號, Q1...Q16`。
- Q1 到 Q16 已是 **1 到 7 的實際同意分數**，不是 Excel 裡原本的選項編號。
- 配對（前後測都做）：**41 人**。
- 只有前測：**S05, S47**。只有後測：**S01, S38, S41**。
- 所有作答都在 **1 到 7** 之間，**沒有缺漏值**。

> 為什麼是 43 / 44 / 41？如果你算出 44 / 45，通常是忘了跳過大標題、或把空白列也算進去了。第一欄的表頭應該是 `帳號`，不是那串標題字。

**Bonus（題 11 的字句）**：第 11 題（ER 構面）在前後測的字句被改寫過，前測偏向「我一定得來參加…」、後測偏向「我必須來上這門課…」。字句變了，前後測就不是量同一件事，嚴格說**不能直接比較**。先記下來，練習 03 會回頭談這個「工具一致性」問題。
