# 練習 02 範例回覆 - 信度分析

> 產出在 [outputs/](outputs/)。

每位學生的構面分數＝該構面四題的平均，已存成 `subscale_scores_pre.csv`（43 列）與 `subscale_scores_post.csv`（44 列），所有分數都在 1 到 7 之間。

Cronbach's alpha 是衡量「同一構面的四題答得一不一致」的標準信度指標，數值介於 0 到 1，越接近 1 代表這幾題越在測同一件事（一般 0.7 以上算可接受）。

**信度表（reliability.csv）**

| 構面 | 前測 alpha | 後測 alpha |
|---|---|---|
| IM | 0.93 | 0.95 |
| IR | 0.90 | 0.96 |
| ER | **0.36** | 0.92 |
| AM | 0.86 | 0.87 |

**低於 0.7 的只有一個：ER 前測 alpha ≈ 0.36**，遠低於可接受門檻。其餘 7 個都在 0.85 以上，很好。

**猜原因**：回想練習 01，第 11 題（ER 的一題）在前後測被改寫過。一題字句不一致、和其他題不同方向，就足以把整個構面的信度拉垮；後測改寫後 ER 就跳回 0.92。

**一句話結論**：這份問卷整體可信（IM/IR/AM 前後測都穩定），但**前測的 ER 不可信，後續凡是牽涉前測 ER 的比較都要特別保留**。

**Bonus（丟掉 Q11）**：把前測 ER 只留 Q3, Q7, Q15（去掉改寫過的 Q11），alpha 會明顯回升，因為 Q11 原本和其他三題是負相關，是它一題在拖累整體。
