# 練習 04 範例回覆 - 畫圖

> 圖在 [outputs/figures/](outputs/figures/)。

四張圖都畫好了，為了避免中文變成空白方框，我直接用英文標籤。每張都加了標題、座標軸標籤，需要的也加了圖例。

| 檔名 | 圖 | 看點 |
|---|---|---|
| grouped_bar.png | 四構面前後測平均並排長條圖 | 四個構面後測都比前測高 |
| box.png | 四構面分數盒鬚圖（前/後） | 看分布與離群值 |
| slope.png | 前後測平均斜率圖（主圖） | 四條線全部上升，一眼看出方向 |
| final_hist.png | 期末成績直方圖 | 60 分及格線；左側有一群 0 分／低分 |

**主圖（hero）是 slope.png**：四個構面的線全部由前測往後測上升。解讀時要分開看：IM/IR 上升是正向訊號，ER 上升代表外在壓力感變強，AM 上升則是無動機增加、需要留意。

> 迭代的重點不是漂亮，而是你看著圖說「這裡看不懂、幫我改」。例如可以再請代理把字放大、換配色，或在斜率圖加一行註解「四構面都上升，但 AM 上升是警訊」。
